Overview Config Commands Help

Essentials

ModLog

This module enables other modules to provide output to a specific channel. In addition to that this module also logs a few events on it's own, including member join, leave, etc. It is recommended though to only give team members access to the mod-log channel.

Commands

This module allows for commands to be run. You may configurate channels, prefix, black/whitelists etc.

Public Servers

Cleanchat

Keeps you chat clean from spam, rude words and harassment. Automatically deletes the message, warns the user and alerts moderators if the mod-log module is active. Users with too many violations will receive a temporary mute, the duration of which increases with every further violation.

Selfroles

Every reaction added within the set channel will add (or remove) the dedicated role.

Dynamic Voices Channels

Always deletes all empty voice channels in the given category but one and gives the first person to join this channel permissions to edit it. After one joins, a new, empty channel will be created.

Other

These modules were designed for a set environment. That being said it's still 100% possible to use those though they will most likely not fit in your server.

Memes

Adds up- and downvote reactions to any images (memes) in the specified memes channel and adds them to a currently private meme database.

Quotes

It's not as cool as it sounds. It allows users to quote other people in a set channel to preserve them for posterity.

Casual

Adding this module is not recommended for public servers, especially not for serious ones. This module gives the bot it's own personality (and mind [in some way])
And yes, it has no properties to set.

Counting

This module moderates a set channel in order to only let users count upwards. Only one text a time and nothing else than texts. Usefulness is questionable.

Tude Club / Private

These modules were designed for private use only. It is unfortunately not possible to use those on your server. If you wish to do so anyways, please contact the bot team. Thank you.
